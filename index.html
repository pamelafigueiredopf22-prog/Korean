한글 Hangul

The Korean alphabet. Tap any letter to hear it.

Korean isn't drawn as a line of letters — sounds stack into little square syllable blocks. A block is a consonant + a vowel: ㄱ + ㅏ = 가 (ga). Learn the pieces and you can read the blocks.

Vowels · 모음

These set the main sound of a block.

Consonants · 자음

These start (and sometimes end) a block.

ㅇ is the quiet one. At the start of a block it makes no sound — it's just a placeholder so a vowel can stand alone: 아 is simply a. At the end of a block it sounds like ng.
